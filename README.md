# AI科学者メンター（仮称）

> 科学技術の偉人と対話し、仕事の姿勢・信念・モチベーションを学ぶアプリ

---

## コンセプト

「ニュートンだったら、この困難をどう乗り越えるか？」

技術系ビジネスマンが日々の仕事で感じる壁——理解されない・先が見えない・モチベーションが落ちる——を、
同じ「技術者」として困難を乗り越えてきた偉人たちの言葉と哲学で突き破る。
